Samuli Cheatsheet

Table Tagit

<Tagi> Nimi Tarkotus
<table> table tablen tekijä, laita tämä ensin
<thead> table head tablen ylin rivi eli otsikot laitetaan tän sisälle
<th> th teksti jota käytetään otsikon sisällä, aina paksu teksti
<tbody> table body tablen body tagi eli kaikki muu paitsi header ja footer
<tr> table row rivi eli rivin sisään tarvitaan vielä data
<td> table data data joka menee tr sisään
<tfoot> table foot tulee viimeisenä

Atrributes

Attribuutti Nimi Tarkotus
colspan column span kuinka monta kolumnia td elementti on
rowspan row span kuinka monta riviä td elementti on
